# Plan 11-01: Abstracted Local Hive Caching Layer

## Objective

Create a centralized, reusable offline caching layer using Hive so that core dashboard data survives network dropoffs. The layer wraps API calls with a "fetch-first, cache-fallback" strategy — if the network call succeeds, the response is cached locally; if it fails with a `SocketException` or `DioException`, the cached version is returned transparently.

## must_haves

- [ ] A `CacheManager` singleton that initializes Hive boxes and provides typed `get`/`set`/`invalidate` methods.
- [ ] Teacher schedule provider fetches from API → caches → falls back to cache on network failure.
- [ ] Parent children provider fetches from API → caches → falls back to cache on network failure.
- [ ] Result analytics provider fetches from API → caches → falls back to cache on network failure.
- [ ] App still loads and displays data when the device is completely offline (using cached data).

<title>Create CacheManager singleton and CacheKeys constants</title>
<read_first>

<action>
1. Create `mobile_app/lib/core/cache/cache_keys.dart`:
   ```dart

     static const String teacherSchedule = 'teacher_schedule';
     static const String parentChildren = 'parent_children';
     static const String resultAnalytics = 'result_analytics';
     static const String feeLedger = 'fee_ledger';
   }
   ```

2. Create `mobile_app/lib/core/cache/cache_manager.dart`:
   ```dart
   import 'dart:convert';
   import 'package:hive/hive.dart';
   
   class CacheManager {
     static const String _boxName = 'dg_smart_cache';
     static Box? _box;
     
     static Future<void> init() async {
       _box = await Hive.openBox(_boxName);
     }
     
     static Future<void> put(String key, dynamic data) async {
       final box = _box ?? await Hive.openBox(_boxName);
       await box.put(key, jsonEncode(data));
     }
     
     static T? get<T>(String key) {
       final box = _box;
       if (box == null) return null;
       final raw = box.get(key);
       if (raw == null) return null;
       return jsonDecode(raw) as T;
     }
     
     static Future<void> invalidate(String key) async {
       final box = _box ?? await Hive.openBox(_boxName);
       await box.delete(key);
     }
     
     static Future<void> clearAll() async {
       final box = _box ?? await Hive.openBox(_boxName);
       await box.clear();
     }
   }
   ```

3. Call `await CacheManager.init();` in `main.dart` after `Hive.initFlutter()`.
</action>

- CacheManager has methods: `init`, `put`, `get`, `invalidate`, `clearAll`

</acceptance_criteria>
